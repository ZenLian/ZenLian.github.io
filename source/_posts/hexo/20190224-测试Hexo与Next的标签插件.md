# Backtick Code Block

效果与Code Block相同，但用的是反引号，与Markdown一致。

```cpp hello.cpp https://en.wikipedia.org/wiki/C++ c++-wiki
#include <iostream>

int main(void)
{
    std::cout << "hello!" << std::endl;

    return 0;
}
```

其他还有`Image`用于插入指定大小的图片、`YouTube`或`Vimeo`用于插入视频、`post_link`用于插入指向其他文章的链接等，就不一一列举了。
